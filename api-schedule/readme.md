# Здесь будет файлы к REST API Schedule (Расписание)

## Требования к программному продукту

- способность запускаться на UNIX-системах и Windows 7, 8, 10;
- поддержка асинхронных HTTP – запросов;
- наличие Gateway - токена для защиты от посторонних приложений и людей;
- реализация HATEOAS;
- репликация данных из других сервисов;
- реализация под возможность MAJOR-обновлений;
- наличие базы данных;
- сервис должен автоматически заполнять атрибуты
- при обращении к доменному имени должен возвращаться список всех возможных ссылок.
